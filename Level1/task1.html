Building my first styled webpage with HTML & CSS

This page is Level 1 - Task 1 of the Primeor Solutions Web Development Internship: a heading, paragraph, image, spacing, a working form, and a responsive navigation menu with hover effects.

About this task

The goal of this task is to practice the fundamentals of front-end development: structuring content with semantic HTML, applying spacing and background colors with CSS, and making a page that feels clean and user-friendly on any screen size. Resize the browser window to see the navigation menu turn into a hamburger menu on smaller screens.
