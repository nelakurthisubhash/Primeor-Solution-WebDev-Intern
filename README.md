📝 Task Details
Level 1 — Task 1: HTML & CSS

A styled webpage built with semantic HTML and CSS, including:

Heading, paragraph, and image with background colors and spacing
A clean form with an input field, submit button, and basic styling
A responsive navigation menu (Home / About / Contact) with hover effects, alignment styling, and a hamburger menu on smaller screens
Level 1 — Task 2: JavaScript

Three small interactive JavaScript features:

An interactive button that changes its background color on click
A dynamic greeting ("Good Morning / Afternoon / Evening") based on the user's current system time
A simple calculator that adds two numbers with input validation
Level 2 — Task 1: Front-end Frameworks

Built using Bootstrap 5:

A responsive card component with an image and text
A responsive grid layout (4 columns → 2 → 1 depending on screen size)
A responsive "Services" section styled like a real company website
Level 2 — Task 2: Responsive Design
A basic webpage made fully responsive using CSS media queries
A mobile-friendly navigation menu that collapses into a hamburger menu on smaller screens
Layouts optimized separately for mobile, tablet, and desktop screens
Level 3 — Task 1: Image Manipulation
A thumbnail image gallery that enlarges images in a lightbox on click
An automatic image slideshow that transitions between images every few seconds, with clickable dot navigation
Smooth hover effects and transitions applied to images
Level 3 — Task 2: Landing Page

A complete landing page for the "Web Developer Internship Program at Primeor Solutions", including:

Clear headline and program introduction
Highlights section (real-world experience, portfolio development, practical frontend exposure, hands-on projects)
Qualifications / desired skills section
Call-to-action buttons ("Apply Now", "Learn More")
Footer with website, email, LinkedIn, and location details
